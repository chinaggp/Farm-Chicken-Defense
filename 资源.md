# AI 农场小游戏美术资源生产规范（适用于抖音 / 微信小游戏）

------

# 1. 项目目标

项目类型：

```text
Save The Pets 类农场防御小游戏
```

核心特点：

- 轻量级
- 强视觉反馈
- 可快速扩展关卡
- AI 批量生成资产
- 统一卡通风格
- 适合广告变现

------

# 2. 推荐整体画风

## 推荐风格（非常适合小游戏）

```text
Disney Cartoon + Casual Mobile Game
```

特点：

| 项   | 方案                 |
| ---- | -------------------- |
| 风格 | 卡通农场             |
| 色彩 | 高饱和暖色           |
| 光影 | 柔和阳光             |
| 线条 | 少描边               |
| 材质 | 绘本质感             |
| 角色 | 圆润可爱             |
| UI   | 木质 + 金色          |
| 场景 | 农场 / 草地 / 木栅栏 |

# 19. 最重要的原则

## 不要追求“大而全”

小游戏核心：

```text
快速上线验证收益
```

先完成：

- 30 关
- 基础美术
- 基础广告
- 基础留存

比：

```text
做 1 年美术
```

更重要。
